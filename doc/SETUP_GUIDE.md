## 필수 요구 사항
- **운영체제**: Windows 10 또는 Windows 11
- **소프트웨어**: [AutoHotkey v2.0 이상](https://www.autohotkey.com/download/ahk-v2.exe)
  - *주의: v1.1(Classic) 버전과는 호환되지 않습니다.*

## 설치 단계

### 1. AutoHotkey 설치
AutoHotkey 공식 웹사이트에서 v2.0 버전을 다운로드하여 설치합니다. 이미 설치되어 있다면 이 단계를 건너뛰십시오.

### 2. 스크립트 파일 다운로드
`controlVolumeByMouse.ahk` 파일을 프로젝트 저장소에서 다운로드하여 원하는 폴더에 저장합니다.

### 3. 스크립트 실행
다운로드한 `.ahk` 파일을 더블 클릭하여 실행합니다. 시스템 트레이(시계 옆)에 AutoHotkey 아이콘(H 모양)이 나타나면 정상적으로 실행된 것입니다.

### 4. 윈도우 시작 시 자동 실행 설정 (권장)
매번 수동으로 실행하지 않으려면 다음 과정을 따르세요:
1. `Win + R` 키를 눌러 실행창을 엽니다.
2. `shell:startup`을 입력하고 확인을 누릅니다.
3. 열린 시작프로그램 폴더에 `controlVolumeByMouse.ahk` 파일의 **바로가기**를 복사해 넣습니다.

## 사용 방법

- **볼륨 조절**: 마우스 포인터를 화면 하단(또는 측면)의 **작업 표시줄** 위로 이동시킨 후, 마우스 휠을 위/아래로 굴립니다.
- **수동 새로고침**: 모니터를 새로 연결했거나 작업 표시줄 위치가 바뀌었는데 인식이 안 될 경우, `F12` 키를 누르면 영역 정보를 다시 수집합니다.

## 문제 해결

### 볼륨 조절이 작동하지 않나요?
1. AutoHotkey v2.0이 설치되어 있는지 확인하세요.
2. 시스템 트레이에 아이콘이 있는지 확인하세요 (없다면 파일을 다시 실행하세요).
3. `F12` 키를 눌러 작업 표시줄 정보를 새로고침해 보세요.
4. 다른 앱이 작업 표시줄 위에서 실행 중(전체 화면 모드 등)인지 확인하세요.

### 특정 프로그램 위에서만 작동안하나요?
관리자 권한으로 실행되는 프로그램 위에서 볼륨을 조절하려면, 이 스크립트 역시 **관리자 권한으로 실행**해야 합니다. `.ahk` 파일을 우클릭하고 '관리자 권한으로 실행'을 선택하세요.
